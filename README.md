# rpos_demo

TPS - RPOS project.

# Install
Before run app, open terminal and run this command first
```bash
flutter pub run build_runner build  --delete-conflicting-outputs
```

## Usage

* Run dev environment
    ```shell
    flutter run lib/main.dart
    ```
